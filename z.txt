# Structure des fichiers CSS et instructions d'intégration

J'ai réorganisé votre code CSS en plusieurs fichiers thématiques pour une meilleure lisibilité et maintenance :

## Fichiers CSS

1. **base.css** - Styles fondamentaux
   - Reset CSS
   - Styles du body
   - En-tête et navigation
   - Pied de page

2. **form.css** - Styles des formulaires  
   - Conteneur principal
   - Étapes de progression
   - Champs de formulaire
   - Boutons
   - Validation et messages d'erreur

3. **alert.css** - Styles de l'alerte de phishing
   - Modal d'alerte
   - Animation
   - Données capturées
   - Conseils anti-phishing

## Comment intégrer les nouveaux fichiers

Pour utiliser cette nouvelle structure, vous devez modifier le fichier `index.html` pour charger les nouveaux fichiers CSS au lieu de l'ancien.

Remplacez :
```html
<link rel="stylesheet" href="styles/main.css">
```

Par :
```html
<link rel="stylesheet" href="styles/base.css">
<link rel="stylesheet" href="styles/form.css">
<link rel="stylesheet" href="styles/alert.css">
```
